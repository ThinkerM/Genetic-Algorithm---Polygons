Detailed Description

Visualization and customization of a Genetic algorithm with Polygons.

Provides following features (among others):

  • Import previously created individuals into a GA's initial population
  • Modify parameters of the genetic algorithm
  • Change frequency of generations in the GA to be viewed
  • Change frequency of generations to be saved during the run of the GA
  • View generated populations and change basic properties of the images
